# Roadmap (2 weeks)

## Week 1
- Утверждение UX-flow и визуального стиля (по вашим референсам).
- Финализация API/БД.
- Подготовка инфраструктуры Yandex Cloud dev.
- Каркас KMM + backend skeleton.
- Auth и ролевая маршрутизация.
- Client dashboard.
- Базовые расчеты тарифов и статусов оплаты.
- Admin list + создание клиента.
- Привязка велосипеда и weekly rate.

## Week 2
- Карточка клиента + история аренд.
- Редактирование комментариев и ссылок.
- Интеграция YooKassa (create payment + webhooks + idempotency).
- Проводки ledger.
- Debt adjustments.
- Полировка edge-cases по расчетам.
- Тестирование (unit + integration).
- Beta rollout.
- Bugfix и релиз-кандидат.
